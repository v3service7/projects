2. Paste the code to your page

  • In your wordpress Admin, go to the page where you want to place the order button

2. Drag & drop an Embed Code element

  • Log in to your Weebly account
  • Decide on where to place the order button
  • Drag an Embed Code element and drop it where you want your button to appear
The more visible the better - make sure your button is high up in the page, so your customers can easily see it.

You cannot embed code into the Weebly template headers, so try to place the button right after the fixed elements of your theme.

2. Add an HTML Code container to your page

  • Log in to your WIX website
  • Go to the editor's left menu and click Add --> More --> HTML Code. A new grey container will appear on the screen.

2. Select the page where the button should appear

  • Log in to your Squarespace account
  • Decide on where to place the order button
  • Insert a "Code" content block
Note: You can add content blocks only in the Page Content section. Make sure you place the button as high as possible, preferably above the fold.

2. Choose the placement of the ordering button

  • Log in to your Drupal account
  • Decide where you want to have the ordering button displayed. Create a new block or simply update an existing one.

2. Check your Joomla settings

  • Log in to your Joomla account and go to your website
  • In the top navigation, go to System ? Global Configuration
  • Next to Default Editor, make sure you have Editor - CodeMirror selected

2. Decide where the button should appear

  • Log in to your Jimdo account
  • Decide on where to place the order button
  • Click on Add element and then Additional elements to see the extended list
  • Select Widget / HTML
